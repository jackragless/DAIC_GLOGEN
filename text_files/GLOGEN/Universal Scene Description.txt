===DOCUMENT BODY===

Universal Scene Description|1| (USD) is a framework for interchange of 3D computer graphics data, created by Pixar. The framework focuses on collaboration, non-destructive editing, and enabling multiple views and opinions about graphics data.File formats|2| used by the standard include:
.usd, which can be either ASCII|3| or bin|4|ary-encoded
.usda, ASCII encoded
.usdc, binary encoded
.usdz, a package file which is a zero-compression, unencrypted zip archive, which may contain usd, usda, usdc, png, jpeg, m4a, mp3, and wav files.


== Support ==
Apple support .usdz files in SceneKit, for 3D model interchange purposes.
Nvidia has announced support for USD|5| in Omniverse, a graphics collaboration platform.
Houdini|6| includes an implementation of USD, for purposes of format interchange and scene editing.
Blender|7| includes support for USD export.
